# 🛠️ Hands-On Labs & Exercises

---

## Coding Challenges

### System Design Coding
1. **Implement a Rate Limiter** — Token bucket with Redis
2. **Implement Consistent Hashing** — Ring-based with virtual nodes
3. **Implement a LRU Cache** — HashMap + Doubly Linked List
4. **Implement a Distributed Lock** — Redis SETNX with TTL
5. **Implement a Circuit Breaker** — State machine (CLOSED/OPEN/HALF_OPEN)

### Spring Boot Labs
1. Build a REST API with pagination, filtering, sorting
2. Implement Kafka producer/consumer with error handling
3. Add Redis caching with cache-aside pattern
4. Implement JWT authentication from scratch
5. Add Resilience4j circuit breaker to external API calls

### Kubernetes Labs
1. Deploy a Spring Boot app to Minikube
2. Set up HPA (Horizontal Pod Autoscaler)
3. Implement rolling update with zero downtime
4. Configure Ingress with TLS
5. Set up Prometheus + Grafana monitoring

### React Labs
1. Build a real-time dashboard with WebSocket
2. Implement infinite scroll with virtualization
3. Build a micro-frontend with Module Federation
4. Optimize a slow React app (profiling + fixes)
5. Implement optimistic updates with TanStack Query

### AI/LLM Labs
1. Build a RAG chatbot with LangChain + pgvector
2. Implement streaming responses with SSE
3. Add function calling to an LLM agent
4. Build a document ingestion pipeline
5. Implement semantic search with embeddings

---

## How to Use
1. Pick a lab matching your target role
2. Time-box to 2-4 hours
3. Focus on production-quality code (error handling, logging, tests)
4. Document your design decisions
5. Review against the solution (if provided)
