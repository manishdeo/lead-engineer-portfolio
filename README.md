## 🚀 How to Use This Repository

### For Quick Preparation (1-2 weeks)
1. Start with [30-Day Plan](./06-preparation-strategies/30-day-plan.md) (accelerated mode)
2. Focus on [Interview Questions Bank](./04-interview-questions/README.md)
3. Review [System Design](./02-system-design/README.md) top 5 systems
4. Practice [Answering Framework](./06-preparation-strategies/answering-framework.md)

### For Comprehensive Preparation (1-2 months)
1. Follow [30-Day Plan](./06-preparation-strategies/30-day-plan.md) systematically
2. Build 2-3 [Portfolio Projects](./03-portfolio-projects/README.md)
3. Master all [Core Technologies](./01-core-technologies/README.md)
4. Complete [System Design Mastery](./02-system-design/README.md)
5. Review [Latest Trends](./07-latest-trends/README.md)

### For Specific Roles
- **Backend Heavy**: Focus on Java, Spring Boot, Microservices, System Design
- **Full Stack**: Balance Java + React + System Design + Cloud
- **Cloud Architect**: AWS, Kubernetes, Distributed Systems, System Design
- **AI Engineer**: GenAI, RAG, LLM + Backend + System Design
- **Engineering Manager**: Leadership + Architecture + All Technical Areas
